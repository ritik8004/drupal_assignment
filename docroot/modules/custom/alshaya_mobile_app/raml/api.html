/product/search get

get

get /product/search

  • Request
  • Response

Query Parameters

  • q: required (string)

    The searched text.

    Example:

    q=dress
  • f: (array of string)

    A list of applied filters.

    Example:

    f[0]=plp_size:M&f[1]=skus_sku_reference_final_price:0
  • sort: (string)

    A sorting option to apply.

    Example:

    sort=final_price DESC

HTTP status code 200

Body

Media type: application/json

Type: object

Properties
  • facets: required (array of Facet)

    Items: Facet

    • key: required (string)

      The technical ID of the filter.

    • label: required (string)

      The human readable value of the filter.

    • options: required (array of items)

      Items: items

      • key: required (string)

        The technical ID of the filter option.

      • label: required (string)

        The human readable value of the filter option.

      • count: required (integer)

        The number of products matching the filter option.

      • extensions: (object)

        An arbitrary object to handle filter specific attribute (color).

  • products: required (array of integer)

    A list of product IDs.

          • alt: (string)

        https://domain.com/path/to/image.png
      • title: (string)
      • alt: (string)
    • assets: (array of items)

      Attribute specific to H&M, contains a list of pre-calculated images for plp and pdp.

/categories get

get

Get all the categories.

get /categories

Get all the categories.

  • Response

HTTP status code 200

Body

Media type: application/json

Type: array of Category

Items: Category

  • id: required (integer)

    The Drupal internal term ID.

  • name: required (string)

    Example:

    Ladies
  • promo_banner: (object)

    It will be generated from the promo_banner, promo_banner_mobile and disable_promo_banner_on_mobile fields.

    • url: required (string)

      Example:

      https://domain.com/path/to/image.png
    • title: (string)
    • alt: (string)
  • path: required (string)

    Example:

    /ladies
  • swatch_type: (string)

    Example:

    still_life
  • quicklink_parent_plp: required (boolean)
  • quicklink_dept_dropdown: required (boolean)
  • include_in_menu: required (boolean)
  • thumbnail: (object)
    • url: required (string)

      Example:

      https://domain.com/path/to/image.png
    • title: (string)
    • alt: (string)
  • image: (object)
    • url: required (string)

      Example:

      https://domain.com/path/to/image.png
    • title: (string)
    • alt: (string)
  • size_guide_anchor: (string)

    Given the size guide is one unique page for all size guide types (shoes, pants, shirts, UE, US, ...), anchor is used to switch directly to proper section when opening size guide.

  • department_page_id: (integer)

    The Drupal internal node ID of related department page if any.

  • child: (array of String)

    An array of category entity - displayed as string[] in documentation to avoid circular dependency.

Example:

{
  "id": "1234",
  "name": "Ladies",
  "promo_banner": {
    "url": "https://domain.com/path/to/ladies/category/promo.png",
    "title": "Ladies",
    "alt": "Ladies"
  },
  "path": "/ladies",
  "swatch_type": "still_life",
  "size_guide_anchor": "ladies",
  "department_page_id": "4567",
  "child": [
    {
      "id": "2222",
      "name": "Pants",
      "path": "/ladies/pants",
      "size_guide_anchor": "ladies_pants"
    },
    {
      "id": "3333",
      "name": "Dresses",
      "path": "/ladies/dresses",
      "size_guide_anchor": "ladies_dresses"
    }
  ]
}
